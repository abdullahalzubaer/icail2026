# GradeLegal: Automated Grading for German Legal Cases

This Repository accompanies the ICAIL 2026 paper titled "GradeLegal: Automated Grading for German Legal Cases".

>Repository Structure:

# Data


- `./data/models-output-from-our-exp` contains the output of the models used in the experiments.

# Prompts

- `./gorp-prompts` contains all the prompts skeleton used in the experiments for Public Law (GORP). Please read the `README.md` file in the `prompts` folder for more information about the prompts.
