## Prompts (Public Law)

This repository includes the **prompt templates (skeletons)** used in our public law experiments.

- The skeletons omit the full prompt text. They are provided to document the **overall structure** of the prompts and how the components are organized.
- The complete prompt text is available in `./code/inference/full_prompts`.

### Language Versions

- **German prompts**: the original prompts used in our experiments. `/prompts/task_public_law/de/`
- **English prompts**: translations of the German prompts, provided **for reference only** (not used in the experiments). `/prompts/task_public_law/en/`
